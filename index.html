Cosmic Cow Hunt

``` Этот код реализует: 1. Главное меню с анимированным космическим фоном 2. Игровой процесс с космическими коровами 3. Реалистичный дизайн коров: - Белое тело с анимированными ногами - Прозрачный синий шлем с рожками - Детализированные глаза и нос 4. Космический фон с движущимися звездами 5. Систему захвата коров при клике 6. Таймер и счетчик очков 7. Адаптивность под разные размеры экрана Чтобы развернуть игру: 1. Сохраните код в файл index.html 2. Загрузите его на GitHub Pages 3. Откройте в Telegram Web Apps Игра полностью работает в браузере и не требует дополнительных ресурсов. Все графика и анимации реализованы через Canvas API. Для улучшения производительности можно добавить оптимизации и более детальные спрайты.
